11. 특정형식을 갖춘 문자열 안에 값 삽입
```
%s   <<<---- 지정한 자리에 문자열 삽입
Ex)
System.out.print("이름: %s", "홍길동");
--> 이름 : 홍길동

%d  정수 값을 10진수로 삽입
%x  정수 값을 16진수로 삽입
%c  정수 값을 문자로 만들어 삽입   65 --> A
%b true/flase 값을 삽입

%[n$]s   n은 무자열에 삽입될 값의 순서+
```
